## 检查生产服务器状态计划

### 1. 检查服务状态
- 检查 PM2 后端服务状态 (`pm2 status`)
- 检查 Nginx 服务状态 (`systemctl status nginx`)
- 检查端口监听情况 (`netstat -tulpn | grep -E ':(3000|80)'`)

### 2. 测试 API 健康检查
- 测试后端直接访问 (`curl http://localhost:3000/health`)
- 测试 Nginx 代理访问 (`curl http://120.26.50.152/api/health`)
- 测试前端页面访问 (`curl http://120.26.50.152`)

### 3. 检查日志文件
- 查看后端错误日志 (`pm2 logs webnote-backend --lines 50 --nostream`)
- 查看后端输出日志
- 查看 Nginx 错误日志

### 4. 检查配置文件
- 检查 Nginx 配置是否存在
- 检查后端环境变量配置
- 检查数据库文件是否存在

### 5. 数据库检查
- 验证 SQLite 数据库文件
- 检查数据库表结构

### 6. 生成状态报告
汇总检查结果，提供当前服务器状态概览和必要的修复建议
